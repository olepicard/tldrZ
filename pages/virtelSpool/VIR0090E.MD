# VIR0090E VIRSV INITIALIZATION ERROR -VSVPOPTR R15 : xxxxxxxx (dddddddd)
`Module`
- 	VIR0000
`Meaning`
- VIRTEL was unable to initialize the VIRSV service environment. xxxxxxxx is the hexadecimal return code from program VSVPOPTR, and dddddddd is the decimal equivalent of the low-order byte of the return code.
`Action`
- Refer to the VIRSV Userâ€™s Guide manual to determine the meaning of the VSVPOPTR return code. Check the VIRTEL log and the VSVTRACE file for additional messages. Check the VIRTEL started task JCL to ensure that the VIRSV load library is referenced in both the STEPLIB and the SERVLIB concatenations. Check that the VIRSV load library is APF- authorized.
